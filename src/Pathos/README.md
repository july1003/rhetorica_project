# Pathos (파토스)

## 이름의 유래

**Pathos**는 그리스어로 '고통', '경험', '감정'을 뜻합니다. 수사학에서는 청중의 **감정에 호소하는 능력**을 의미하며, 텍스트로는 알 수 없는 면접자의 **표정, 태도, 비언어적 뉘앙스**를 심층적으로 분석함을 상징합니다.

## 역할 (Vision & Analysis)

면접자의 **비언어적 요소**와 **심층 정보**를 분석하는 백그라운드 워커입니다.

* **비전 분석**: 얼굴 인식, 표정 변화 감지, 시선 추적 등을 수행합니다.
* **심층 분석**: 이력서와 자기소개서의 맥락을 분석하고, 면접 내용의 정서적 톤을 평가합니다.
* **기술 스택**: OpenCV, DeepFace, Celery Worker, PDF Analysis
