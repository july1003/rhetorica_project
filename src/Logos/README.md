# Logos (로고스)

## 이름의 유래

**Logos**는 그리스어로 '말', '논리', '이성'을 뜻합니다. 아리스토텔레스의 수사학에서 상대를 설득하는 **논리적 근거**를 의미하며, 면접 과정에서 오가는 **언어적 상호작용**과 **지적 흐름**을 상징합니다.

## 역할 (Real-time AI Node)

사용자와의 **소통**을 담당하는 최전선 서버입니다.

* **실시간 대화 처리**: 사용자의 음성을 인식(STT)하고 AI의 응답을 생성(LLM)합니다.
* **논리적 인터페이스**: 면접의 질문과 답변 흐름을 제어합니다.
* **기술 스택**: FastAPI, Redis, STT/TTS 엔진
