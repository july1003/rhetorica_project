# Ethos (에토스)

## 이름의 유래

**Ethos**는 그리스어로 '성품', '관습'을 뜻하며 '윤리(Ethics)'의 어원입니다. 수사학에서는 화자의 **신뢰성**과 **권위**를 의미합니다. 시스템의 모든 데이터가 모이는 곳이자, 변하지 않는 **신뢰의 원천(Source of Truth)**임을 상징합니다.

## 역할 (Central Data)

시스템의 **데이터 존속성**과 **인프라**를 담당하는 핵심 저장소입니다.

* **데이터 통합**: 모든 면접 기록, 사용자 정보, 분석 결과를 저장합니다.
* **신뢰성 보장**: PostgreSQL(벡터/관계형)과 Oracle(엔터프라이즈)을 통해 데이터 무결성을 관리합니다.
* **기술 스택**: PostgreSQL, Oracle, Docker Volume Management
