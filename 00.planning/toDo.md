1. 3개의 서버(docker 이용)

기능

* 전제 : 응시자는 최초 N번은 무료로 면접 가능, default = 3회
* 모의면접시스템 회원 가입 (이름, 이메일, 비번, 지원분야(직업군), 지원회사1,2,3 : 3개 까지 지원가능)
* 이메일 인증
* 로그인
* 메인 : 지원분야 및 지원회사 소식 ( 비 로그인시 : 서비스 소개 페이지)
* 모의 면접이 가능한 회사 또는 분야 선택
* 회사 정보 출력
  1. 인재상
  2. 채용정보
* 이력서 업로드 : (PDF) - MINIO 설치된 Ethos resumes 바스켓에 일자별 폴더안에 면접id_resume_{timestamp}.pdf 로 저장
  => 이력서 분석하여 ai 면접관 3명 중 한명 할당
* 영상 면접 실시
* 면접 종료 후 모의 면접 이력 목록 출력(면접일시, 응시자ID, 응시자이름, 면접시간, 면점점수, 결과)
  1. 모의면접 점수 그래프 출력
  2. * 모의 면접 상세(면접일시, 면접관 총평)
     * 모의 면접 결과
       1. 강점 설명
       2. 약점 설명
  3. 그래프 출력
* 자기소개서 첨삭 지도
