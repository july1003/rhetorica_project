# Pathos (파토스) - Vision & Analysis Node

## 이름의 유래

**Pathos**는 '경험', '감정', '정서'를 뜻하는 그리스어입니다.
수사학에서 **청중의 감정에 호소**하는 것을 의미하듯, 이 서버는 텍스트 너머에 있는 면접자의 **비언어적 요소(표정, 목소리 톤)**와 **심층적인 맥락**을 읽어내고 분석하는 감성/심층 분석 역할을 맡고 있어 붙여진 이름입니다.

## 주요 역할

무거운 연산이 필요한 **비동기 분석 작업**과 **미디어 처리**를 담당합니다. 실시간 응답보다는 정확하고 깊이 있는 분석에 초점을 맞춥니다.

## 포함된 서비스 (Service Stack)

이 폴더의 `docker-compose.yml`은 다음 컨테이너들을 실행합니다:

1. **`analysis_worker`**:
    * **역할**: 얼굴 인식, 표정 분석, 동영상 처리 등 고부하 AI 작업을 수행합니다.
    * **큐**: `analysis_queue`
    * **자원**: GPU 사용

2. **`tts_service` (Coqui TTS)**:
    * **역할**: 텍스트를 감정이 담긴 음성으로 변환해줍니다.
    * **포트**: 5002
    * **자원**: GPU 사용

3. **`batch_worker`**:
    * **역할**: 주기적이거나 대량의 데이터를 처리하는 배치 작업을 수행합니다.
    * **큐**: `batch_queue`

4. **`celery_beat`**:
    * **역할**: 정해진 시간에 작업을 예약하고 실행하는 스케줄러입니다.

## 실행 방법

```bash
docker-compose up -d --build
```
