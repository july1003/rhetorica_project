# Logos (로고스) - Real-time AI Node

## 이름의 유래

**Logos**는 '말', '논리', '이성'을 뜻하는 그리스어입니다.
아리스토텔레스의 수사학에서 **논리적 설득**을 의미하며, 이 서버는 사용자와의 **실시간 대화(언어)**와 **논리적 흐름**을 담당하는 최전선(Front-line)이기 때문에 붙여진 이름입니다.

## 주요 역할

사용자와 직접 상호작용하는 **실시간 서비스 그룹**입니다. 응답 속도가 중요하며, 대화의 논리적 흐름을 제어합니다.

## 포함된 서비스 (Service Stack)

이 폴더의 `docker-compose.yml`은 다음 컨테이너들을 실행합니다:

1. **`web` (FastAPI)**:
    * **역할**: 메인 API 서버. 클라이언트 요청을 받아 LLM이나 워커에게 작업을 분배합니다.
    * **포트**: 8000
    * **특징**: `src` 폴더가 마운트되어 있어 코드 수정 시 즉시 반영됩니다.

2. **`rt_worker` (Real-time Worker)**:
    * **역할**: 실시간성이 요구되는 가벼운 AI 작업(STT, 간단한 텍스트 처리)을 처리합니다.
    * **자원**: GPU 사용

3. **`ollama` (LLM Server)**:
    * **역할**: 채팅 및 면접 질문 생성을 위한 거대언어모델(LLM)을 구동합니다.
    * **포트**: 11434
    * **자원**: GPU 사용

4. **`redis`**:
    * **역할**: 서비스 간 메시지 브로커 및 캐시 저장소.
    * **포트**: 6379

5. **`phoenix`**:
    * **역할**: LLM 실행 추적 및 관측(Observability).
    * **포트**: 6006

## 실행 방법

```bash
docker-compose up -d --build
```
